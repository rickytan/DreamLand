# DreamColor
## 说明
本应用是一款着眼睡眠的健康类产品，与 **[SleepCycle]()** 类似，应用检测用户的动作并决定最终的闹钟响起时间（没有完全实现），同时另外引入了新的元素：灯光。

本应用是 *浙大工业设计* 的毕业设计作品，产品功能部分实现，是个原型产品
